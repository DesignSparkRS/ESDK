Introduction
------------

The Main board plugs into a Raspberry Pi SBC and provides:

* Touch screen
* Four physical buttons
* Battery backed real-time clock (RTC)
* Buzzer
* Sensor chain 3v3 power supply
* Connector for cable to :doc:`pea`
